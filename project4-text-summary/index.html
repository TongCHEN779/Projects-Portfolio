Summarise a pile of messages with k-center

Paste some text. We split it into units — individual words (for keywords) or sentences (split on commas, not periods) — embed each as a vector, then run greedy k-center: the farthest-first traversal that keeps picking the unit least like everything chosen so far. The result is a coreset — k maximally diverse keywords or sentences that summarise the whole pile. k-center chases the most different units, which on sequential text tends to surface outliers — so you can switch the selector to MMR (relevance − redundancy), facility-location (submodular coverage), or LexRank (graph centrality), and tick reading order to list the picks in their original sequence. Core summary

How it works. Each message becomes a TF-IDF vector (words weighted by how distinctive they are), so distance = 1 − cosine similarity. Greedy k-center starts from the most central message, then repeatedly adds the message farthest from everything picked so far. That farthest-first rule is a classic 2-approximation to the k-center objective (minimise the largest distance from any message to its nearest representative) — so the picks spread out to cover every theme. The keywords under each pick are that representative's highest-weight TF-IDF terms. All client-side.

Select vs. synthesize. The coreset picks existing units (k-center). The synthetic ones are generated by sparse reconstruction: each cluster has a centroid — the mean of its members, a new point that's no original message — and matching pursuit greedily stitches together short phrase-fragments drawn from the cluster that best reconstruct that centroid. The result recombines the cluster's wording into a sentence none of them actually wrote (the … marks where fragments were joined). Fully fluent grammar would still want a language model on top; here the geometry + greedy reconstruction already generate a synthetic summary.

Where this is useful. Distilling a long thread, a feedback dump, or a channel into a few representative messages, keywords, and synthesized themes — preserving the diversity of what was said, not just the most frequent words.
